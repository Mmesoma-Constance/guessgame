Guess My Number Game

Game Rules

  • Objective : Your objective is to guess the secret number chosen by the game.
  • Gameplay : - The game randomly select a secret number between 1 and 50.
    - Erase the previous guess if necessary and enter a new number to continue your attempts to guess the secret number.
    - You have a limited time to guess the secret number correctly.
    - After each guess, you can click the "Again!" button to reset and generate a new secret number so as to secure 3 wins.
  • Winnig : To win the game, you need to achieve 3 wins before the time runs out, and be crowned "Game Master".
  • Have Fun : Enjoy the challenge and excitement of trying to guess the secret number against the clock!
